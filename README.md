# 中文语言腐败词库 (Chinese Language Corruption Lexicon)

本项目旨在系统性地收集和整理中文互联网上的“语言腐败”现象，按照不同类别归纳为词条和句式。其主要目的是为了帮助用户察觉网络上存在的有害表达，筛选掉垃圾信息（包括意识形态宣传、歧视性用语、脏话、情绪操控等），并为未来开发配套的浏览器内容过滤插件提供标准规则库。

## 分类体系

本项目将收集的词汇和句式分为以下几个主要类别，每个类别对应一个独立的 Markdown 文件：

1. **`01_internet_slang.md` (网络烂梗/空洞词汇)**
   - 滥用的网络流行语，缺乏实际语义。
   - 示例：yyds、绝绝子、家人们。
2. **`02_clickbait.md` (标题党/情绪操控)**
   - 用于制造恐慌、夸大事实、挑动情绪的词汇。
   - 示例：震惊、重磅、炸了。
3. **`03_propaganda.md` (煽动宣传/意识形态)**
   - 带有明显政治操控目的、煽动对立或极端情绪的话术。
   - 示例：斩杀线。
4. **`04_discrimination.md` (歧视与偏见)**
   - 对特定民族、地域、性别、群体等的蔑称或偏见表达。
   - 示例：小日子、米国。
5. **`05_profanity.md` (脏话/低俗用语)**
   - 侮辱性、低俗的词汇。
6. **`06_manipulative_patterns.md` (特定句式/话术)**
   - 固定的操控性表达模式，通常不仅仅是单个词汇，而是特定的句子结构。
   - 示例：“不转不是中国人”。

## 规则格式设计

为了便于未来配套浏览器插件解析，本项目采用一种简单、易读的规则格式。每个类别文件中的规则以列表形式呈现，每行代表一条规则。

### 基本格式

每条规则遵循以下基本格式：

```
[匹配类型] 规则内容 | 权重 | 备注说明
```

### 字段说明

- **[匹配类型]**：指示如何匹配该规则。
  - `[W]` (Word): 精确词汇匹配。
  - `[P]` (Phrase/Pattern): 短语或句式匹配（支持简单的通配符 `*` 代表任意字符）。
  - `[R]` (Regex): 正则表达式匹配（适用于复杂句式）。
- **规则内容**：需要屏蔽或标记的具体词汇或句式。
- **权重** (可选，默认为 1)：表示该词条的严重程度或屏蔽优先级（1-5，5 为最高）。
- **备注说明** (可选)：对该词条的解释、来源或示例。

### 示例

```markdown
[W] yyds | 1 | 网络滥用缩写，缺乏实质内容
[W] 小日子 | 4 | 对日本人的歧视性蔑称
[P] 震惊！* | 3 | 典型标题党句式
[R] ^不转不是.*人$ | 5 | 道德绑架与情绪勒索句式
[W] 斩杀线 | 4 | 煽动性宣传话术
```
