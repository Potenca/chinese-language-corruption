# 中文语言腐败词库 (Chinese Language Corruption Lexicon)

本项目旨在系统性地收集和整理中文互联网上的“语言腐败”现象，按照不同类别归纳为词条和句式。其主要目的是为了帮助用户察觉网络上存在的有害表达，筛选掉垃圾信息（包括意识形态宣传、歧视性用语、脏话、情绪操控等），并为未来开发配套的浏览器内容过滤插件提供标准规则库。

## 分类体系

本项目将收集的词汇和句式分为以下几个主要类别，每个类别对应一个独立的 Markdown 文件：

1. **`01_internet_slang.md` (网络烂梗与空洞词汇)**
   - 滥用的网络流行语，缺乏实际语义，导致表达能力退化。
   - 示例：yyds、绝绝子、家人们。
2. **`02_clickbait.md` (标题党与情绪操控)**
   - 用于制造恐慌、夸大事实、挑动情绪的词汇。
   - 示例：震惊、重磅、炸了。
3. **`03_propaganda.md` (煽动宣传与意识形态)**
   - 带有明显政治操控目的、煽动对立、民族主义狂热或极端情绪的话术。
   - 示例：斩杀线、境外势力、递刀子。
4. **`04_discrimination.md` (歧视与偏见)**
   - 对特定民族、国家、地域、性别、群体、职业等的蔑称或偏见表达。
   - 示例：小日子、米国、捞女。
5. **`05_profanity.md` (脏话与低俗用语)**
   - 具有明显侮辱性、攻击性或低俗性质的词汇（已剔除仅在成人内容平台合理使用的中性词汇）。
   - 示例：各级粗口及规避审查的谐音。
6. **`06_manipulative_patterns.md` (特定句式与话术)**
   - 固定的操控性表达模式，通过特定的句子结构实现道德绑架、情绪勒索或逻辑谬误。
   - 示例：“不转不是中国人”、“抛开事实不谈”。
7. **`07_corporate_jargon.md` (互联网大厂黑话与商业滥用词)**
   - 源自互联网大厂或商业领域的空洞概念，掩饰内容匮乏。
   - 示例：赋能、抓手、底层逻辑。
8. **`08_platform_slang.md` (平台专属黑话与特有词汇)**
   - 各大互联网平台（微博、B站、小红书、抖音等）内部形成并广泛使用的特有词汇、弹幕梗和社区黑话。
   - 示例：绝绝子、前方高能、夹总。
9. **`09_political_campaigns.md` (政治运动与阶级斗争词汇)**
   - 中国共产党历次政治运动（如土改、镇反、反右、大跃进、四清、文革等）中产生的典型政治帽子、批斗话语和运动黑话。
   - 示例：黑五类、走资派、牛鬼蛇神、大串连。

## 规则格式设计

为了便于未来配套浏览器插件解析，本项目采用一种简单、易读的规则格式。每个类别文件中的规则以列表形式呈现，每行代表一条规则。

### 基本格式

每条规则遵循以下基本格式：

```
[匹配类型] 规则内容 | 权重 | 备注说明
```

### 字段说明

- **[匹配类型]**：指示如何匹配该规则。
  - `[W]` (Word): 精确词汇匹配。
  - `[P]` (Phrase/Pattern): 短语或句式匹配（支持简单的通配符 `*` 代表任意字符）。
  - `[R]` (Regex): 正则表达式匹配（适用于复杂句式）。
- **规则内容**：需要屏蔽或标记的具体词汇或句式。
- **权重** (可选，默认为 1)：表示该词条的严重程度或屏蔽优先级（1-5，5 为最高）。
- **备注说明** (可选)：对该词条的解释、来源或示例。

### 示例

```markdown
[W] yyds | 1 | 网络滥用缩写，缺乏实质内容
[W] 小日子 | 4 | 对日本人的歧视性蔑称
[P] 震惊！* | 3 | 典型标题党句式
[R] ^不转不是.*人$ | 5 | 道德绑架与情绪勒索句式
[W] 斩杀线 | 4 | 煽动性宣传话术
```
